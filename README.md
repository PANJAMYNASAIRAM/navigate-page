##Project Description

This project is a simple and interactive Bookmark Manager Web Application built using JavaScript. It allows users to add website names and URLs, which are then displayed as clickable links.

When a user clicks on a saved bookmark, it opens the corresponding website in a new tab. The application also includes basic form validation to ensure that required fields are filled before submission.

##Features

🔖 Add Bookmarks
Users can enter a site name and URL to save bookmarks 

🔗 Clickable Links
Saved bookmarks open in a new tab when clicked 

⚡ Dynamic Content Update
Bookmarks are added instantly without page reload 

🚫 Form Validation
Displays error messages if fields are empty 

🧹 Auto Clear Inputs
Clears input fields after successful submission

##Technologies Used

HTML5 – Form structure and layout 

CSS3 – Styling and UI design 

JavaScript (ES6) – DOM manipulation and event handling

##Installation Steps

1. Clone the repository git clone https://github.com/PANJAMYNASAIRAM/navigate-page.git
2. Go to project folder "cd navigate-page"
3. Install dependencies "npm install"
4. Start the project "npm start"

##How It Works
User enters website name and URL
Clicks the submit button
Application validates inputs
Creates a clickable link dynamically
Adds the bookmark to the list
Clicking the link opens the website in a new tab
